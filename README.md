## Install opencv4nodejs on Mac OS

### install Command Line Tools
```bash
$ xcode-select --install
```

### install opencv

1. install cmake
```bash
$ brew install cmake
```

2. install opencv
```bash
$ brew install opencv
```

3. install opencv4nodejs
```bash
$ npm install
```
